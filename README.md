🍽️ Food Delivery SQL Analysis Project

📌 Project Overview

This project analyzes an online food delivery dataset using **MySQL** to gain insights into customer behavior, restaurant performance, order trends, and customer ratings. It demonstrates practical SQL skills commonly used in data analytics and business intelligence.

🎯 Project Objectives

* Analyze restaurant performance based on revenue and customer ratings.
* Identify loyal and high-value customers.
* Explore ordering patterns and customer preferences.
* Generate business insights using SQL queries.
* Practice database design, data import, and SQL analysis.

🛠️ Technologies Used

* Database: MySQL
* QL IDE:MySQL Workbench
* Version Control: Git & GitHub
* Dataset Format:CSV

▶️ How to Run the Project

1. Clone this repository.
git clone https://github.com/your-username/Food-Delivery-SQL-Project.git

2. Open MySQL Workbench.

3. Create a new database.

4. Import all CSV files using the **Table Data Import Wizard**.

5. Execute the SQL scripts in the `SQL` folder.

6. View the query outputs and insights.
